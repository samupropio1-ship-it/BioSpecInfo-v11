## PATTERN DI SVILUPPO

### Overlay iframe (pattern standard):
```javascript
window.openTesi=function(){
  var ov=document.getElementById('bsi-tesi-ov');
  if(!ov){
    ov=document.createElement('div');
    ov.id='bsi-tesi-ov';
    ov.style.cssText='position:fixed;inset:0;z-index:9999;background:#071221;display:flex;flex-direction:column';
    var tb=document.createElement('div');
    tb.style.cssText=_BSI_TB_STYLE; // topbar con safe-area-inset-top
    var fr=document.createElement('iframe');
    fr.src='changelog_tesi.html';
    fr.style.cssText='flex:1;border:none;width:100%;height:100%';
    var cb=document.createElement('button');
    cb.setAttribute('style',_BSI_BACK_BTN); // bottom:calc(20px + env(safe-area-inset-bottom,0px))
    cb.innerHTML='↩ Indietro';
    cb.onclick=function(){ov.style.display='none';};
    ov.appendChild(tb);ov.appendChild(fr);ov.appendChild(cb);
    document.body.appendChild(ov);
  } else ov.style.display='flex';
};
```

### SW cache bump (ogni deploy):

### RDKit loading (robusto):
- 6 CDN (jsdelivr + unpkg, 3 versioni) con retry automatico
- Fallback immediato con SmilesDrawer per strutture 2D
- smilesDescHeuristic(smi): stima descrittori da SMILES senza RDKit
- Pulsante "↺ Ricarica" se tutti i CDN falliscono
- initRDKitRobust() / window._rdkInitFn() per retry
