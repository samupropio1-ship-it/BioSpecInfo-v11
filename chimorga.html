1 · Fondamenti della Chimica Organica

1.1 Orbitali ibridi e geometria

sp³ → 4σ. sp² → 3σ + 1π. sp → 2σ + 2π. Più carattere s → legame più corto e forte, H più acido.

1.2 Effetti elettronici

Induttivo (I): si trasmette attraverso legami σ, decade con la distanza. –F, –Cl, –OH, –NO₂ = –I. –CH₃, alchili = +I.

Mesomerico / risonanza (M): delocalizzazione su sistema π. –OR, –NR₂, –OH = +M. –NO₂, –CN, –CHO, –COOR = –M.

Iperconiugazione: orbitali C–H σ leggermente donatori verso orbitali p (stabilizza carbocationi/radicali alchilici).

3 · Stereochimica

3.1 Chiralità e centri stereogenici

Un C è stereogenico se ha 4 sostituenti tutti diversi. Enantiomeri: immagini speculari non sovrapponibili — stesse proprietà fisiche in ambienti achirale ma opposta rotazione ottica. Diastereomeri: stereoisomeri non enantiomeri — proprietà fisiche diverse.

Miscela racemica: 50:50 R+S → [α]mix = 0. ee (eccesso enantiomerico): ee = |R–S|/(R+S) × 100%.

3.2 Regole CIP — Assegnazione R/S

Identifica C stereocentrico (4 sostituenti diversi)
▼
Assegna priorità 1→4 per numero atomico; in caso di parità, espandi fino alla prima differenza
▼
Posiziona il gruppo con priorità più bassa (4) lontano da te
▼
Senso orario (1→2→3)
R (Rectus)
Senso antiorario
S (Sinister)
Regola del doppio legame CIP: C=O → C "legato" a (O,O); O "legato" a (C,C). Quindi O > C in priorità. Per C=C, ogni C conta come se fosse legato al C dell'altro lato.

Con n centri stereogenici: massimo 2ⁿ stereoisomeri, ridotti da simmetria interna (forma meso: piano di simmetria interno → otticamente inattiva).

Acido tartarico: 2R,3R / 2S,3S (enantiomeri) + meso-2R,3S (achiral). = 3 stereoisomeri, non 4.

3.3 Proiezione di Fischer

Catena verticale, C1 in alto. Linee orizzontali = legami verso l'osservatore; verticali = lontano. Scambio di 2 coppie → stessa conf.; scambio di 1 coppia → inverte conf.

COOH | H–C–OH → D-gliceraldeide (R) | CH₂OH

4 · Alcheni e Alchini

4.1 Isomeria E/Z e Markovnikov

E/Z: priorità CIP; E = gruppi ad alta priorità su lati opposti; Z = stesso lato.

Regola di Markovnikov: in addizione di HX, il H va sul C meno sostituito (più povero di H). Spiega via stabilità del carbocatione intermedio.

Anti-Markovnikov (Kharasch): HBr + ROOR → Br sul C meno sostituito, meccanismo radicalico.

4.3 Alchini — acidità e reazioni

pKₐ ~25 degli alchini terminali: C sp ha alta s-character → lega l'elettrone più vicino al nucleo → H più acido degli alcani.

R–C≡C–H + NaNH₂ → R–C≡C⁻ Na⁺ + NH₃ R–C≡C⁻ + R'–X (1°) → R–C≡C–R' (allungamento catena)

Riduzione selettiva: Lindlar → cis-alchene; Na/NH₃ liquida → trans-alchene; H₂/Pd-C → alcano.

Idratazione: H₂O/H₂SO₄/HgSO₄ → chetone (Markovnikov) via enolo intermedio.

6.2 SN2 — Sostituzione Nucleofila Bimolecolare

  • v = k[RX][Nu⁻] — cinetica 2° ordine
  • Inversione di Walden (meccanismo ad ombrello)
  • Favorito da substrati 1° (no impedimento sterico)
  • Solventi aprotici polari (DMSO, DMF, acetone) → nucleofilo libero, più reattivo
  • Nucleofilia in aprotici: I⁻ > Br⁻ > Cl⁻ > F⁻ (polarizzabilità domina)
  • Nucleofilia in protici: F⁻ > Cl⁻ > Br⁻ > I⁻ (solvatazione domina)

6.3 SN1 — Sostituzione Nucleofila Unimolecolare

  • v = k[RX] — cinetica 1° ordine; ionizzazione = stadio lento
  • C⁺ planare sp² → attacco da entrambe le facce → racemizzazione parziale/totale
  • Favorito da substrati 3°, allilici, benzilici
  • Solventi protici polari (H₂O, EtOH) stabilizzano anioni e cationi per solvatazione
  • Possibile riarrangiamento del C⁺ (shift 1,2 o espansione d'anello)

7 · Alcoli, Eteri ed Epossidi

7.1 Proprietà e reazioni degli alcoli

Legami H: punti di ebollizione molto più alti degli alcani analoghi. Solubilità decresce con la catena (C1–C3 miscibili, >C4 parzialmente, >C6 immiscibili).

Ossidazione 1° → Aldeide: PCC, Swern (DMSO/oxalilcloruro), DMP (Dess-Martin)Selettivi: si fermano all'aldeide
Ossidazione 1° → Acido: KMnO₄/H⁺, Jones (CrO₃/H₂SO₄aq)
Ossidazione 2° → Chetone: qualsiasi Cr(VI), Swern, DMP
Alcol 3°: non ossidabile (nessun H su C–OH)
→ Alogenuro: HX (HI>HBr>HCl); SOCl₂/py → RCl con ritenzione; PBr₃ → RBr con inversione
Disidratazione: H₂SO₄ conc./Δ → alchene (Zaitsev) via C⁺3° > 2° > 1°; 1° richiede T alta (170°C)

7.2 Sintesi di Williamson degli eteri

R–OH + NaH → R–O⁻ Na⁺ + H₂↑ R–O⁻ + R'–X (1°) → R–O–R' + X⁻ (SN2)
Usa come nucleofilo l'alcossido dell'alcol meno ingombrato; come elettrofilo l'alogenuro primario. Alogenuri 3° danno eliminazione.

7.3 Epossidi — sintesi e apertura

Sintesi: alchene + mCPBA → epossido (stereospecifica, ritenzione geometria doppio legame)
Apertura basica (Nu⁻, OH⁻): attacco SN2 sul C meno sostituito → prodotto anti Prodotto = trans-diolo o trans-β-idrossiNu
Apertura acida (H⁺/Nu): protonazione O⁻ → carica + più sul C più sostituito → Nu attacca il C più sostituito → anti H₂O/H⁺ → diolo trans; HBr/H⁺ → bromoalcol trans

8.3 Condensazione aldolica

2 CH₃CHO →[NaOH]→ CH₃CH(OH)CH₂CHO (aldolo = β-idrossicarbonile) →[Δ]→ CH₃CH=CHCHO + H₂O (enone α,β-insaturo)
  1. Base rimuove α-H → enolato
  2. Enolato attacca carbonile di 2ª molecola
  3. Protonazione → β-idrossicarbonile
  4. Disidratazione con Δ → enone
Aldolo incrociato: efficace se uno dei due carbonili non ha α-H (formaldeide, benzaldeide) → no auto-condensazione competitiva.

8.4 Test per aldeidi

Tollens (specchio d'argento): RCHO + [Ag(NH₃)₂]⁺ → RCOO⁻ + Ag↓ Diagnosi aldeide. Falso positivo: fenoli, glucosio.
Fehling/Benedict: RCHO + Cu²⁺ → RCOO⁻ + Cu₂O↓ (rosso) Zuccheri riducenti positivi. Chetoni (eccetto α-idrossichetoni) negativi.

9 · Acidi Carbossilici e Derivati

9.1 Scala di reattività

Alogenuri acilici > Anidridi > Esteri ≈ Acidi > Ammidi >> Carbossilato⁻ (+ reattivo → – reattivo) Un derivato più reattivo può formare uno meno reattivo, NON viceversa direttamente.

9.2 Reazioni principali

Esterificazione di Fischer: RCOOH + R'OH ⇌[H⁺] RCOOR' + H₂O Reversibile; spostare equilibrio con eccesso ROH o rimozione H₂O (Dean-Stark)
Saponificazione: RCOOR' + NaOH → RCOO⁻ Na⁺ + R'OH Irreversibile (base forte). Usato industrialmente per saponificazione dei grassi.
Riduzione LiAlH₄: RCOOH, RCOOR', RCONH₂ → R–CH₂OH (alcol primario)
DIBAL-H (1 eq., −78°C): RCOOR' → R–CHO (aldeide, selettivo)
Ammide da alogenuro acilico: RCOCl + R'NH₂ → RCONHR' + HCl

9.3 Hell–Volhard–Zelinskij (HVZ)

R–CH₂COOH + Br₂/P → R–CHBr–COOH + HBr (α-bromo acido — nucleo per SN2/E2 successivi)

9.4 Decarbossilazione β-chetoacidi

R–CO–CH₂–COOH →[Δ]→ R–CO–CH₃ + CO₂ (via TS ciclico a 6 atomi — stato di transizione per-ciclic)

11 · Reazioni di formazione C–C

11.1 Condensazione di Claisen

2 CH₃COOEt →[NaOEt]→ CH₃COCH₂COOEt + EtOH (acetoacetato di etile)
  1. NaOEt rimuove α-H → enolato estere
  2. Enolato attacca C=O di un secondo estere
  3. Espulsione EtO⁻ → β-chetoestere
  4. Il β-chetoestere è abbastanza acido (pKₐ ~11) da essere deprotonato → guida l'equilibrio

Claisen incrociato: usa un estere senza α-H (es. PhCOOEt, EtOOCCOOEt) per evitare auto-condensazione.

11.2 Reazione di Diels–Alder

Diene (s-cis) + Dienofilo → Cicloesene Es: butadiene + etilene → cicloesene butadiene + AN (acrylonitrile) → 4-cianocicloes-2-ene
  • Addizione [4+2] concertata (periodica): syn sulle due componenti
  • Il diene deve essere s-cis (o vincolato in sella)
  • Endo vs exo: endo prevalente (interazioni secondarie HOMO–LUMO)
  • FMO: HOMO(diene) + LUMO(dienofilo). EDG su diene alzano HOMO; EWG su dienofilo abbassano LUMO → gap ↓ → reazione più veloce e selettiva
  • Regiochemistry endo: "orto" e "para" (regola delle componenti equivalenti)

11.3 Reazione di Wittig

Ph₃P + R–X → [Ph₃P⁺–R] X⁻ (sale fosfonio) [Ph₃P⁺–CHR'] + n-BuLi → Ph₃P=CHR' (ilide) Ph₃P=CHR' + R''CHO → R''CH=CHR' + Ph₃P=O
  • Posizione doppio legame definita (no isomeri di posizione)
  • Ilidi non stabilizzate (alchile) → Z (cis) prevalente
  • Ilidi stabilizzate (EWG vicino a C carbanoide) → E (trans) prevalente
  • HWE (Horner-Wadsworth-Emmons) usa fosfonati → più reattivo, E selettivo

11.4 Sintesi malononica e acetacetica

Sintesi malononica: malonato di dietile + NaOEt + R–X → R–CH(COOEt)₂ → [1)NaOH; 2)H⁺; 3)Δ] → R–CH₂–COOH (+1C rispetto a R–X) Sintesi acetacetica: acetoacetato di etile + NaOEt + R–X → R–CH(COCH₃)(COOEt) → [1)NaOH; 2)H⁺; 3)Δ] → R–CH₂–COCH₃ (metilchetone)

11.5 Michael e Robinson

Addizione di Michael: enolato + sistema α,β-insaturo → addotto 1,4 Annellazione di Robinson = Addizione di Michael + Condensazione aldolica + Deidratazione → Anello 6 cicloesenonico nuovo

13.3 Lipidi e acidi grassi

Saturi: catene lineari, impacchettamento ordinato, solidi (grassi animali). Es. palmitico C16:0, stearico C18:0.

Insaturi: doppi legami cis → kink → liquidi (oli vegetali). Oleico C18:1 Δ9-cis; linoleico C18:2 Δ9,12-cis (essenziale).

Saponificazione: trigliceride + NaOH → 3 sali di acido grasso (sapone) + glicerolo.

Fosfolipidi: glicerolo + 2 acidi grassi + fosfato + testa polare → membrana biologica (doppio strato).
